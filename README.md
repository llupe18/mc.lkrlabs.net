# Custom Create Mod SMP

Folgende Anleitung dient zur Installation und verwendung des Custom Mod Packs. Es gibt die Modrinth Variante zum direkten Import oder ein ZIP mit den Mods und Shaderpacks.  

- Minecraft: Neoforge 1.21.1  

> Tipp: Für Schwächere Hardware kann `Distant Horizons` und `Iris Shaders` deaktiviert werden

## Modrinth

1. **Luncher Installieren:**   
https://modrinth.com/app

2. **Modrinth Modpack herunterladen:**  
`Custom Create Mod SMP 1.x.x.mrpack` 

3. **In Modrinth mit einem Minecraft Account anmelden.**

4. **In Modrinth eine neue Instanz hinzufügen:**  
`"➕ (Create new instance)" -> "Install modpack" -> "Import modpack" -> Custom Create Mod SMP 1.x.x.mrpack öffnen`

5. **Spiel Starten:**  
`Play`

6. **Server Hinzufügen:**  
In Minecraft `"Multiplayer" -> "Add Server"` einen Namen Vergeben und die Server Adresse eintragen.

- Server: mc.lkrlabs.net  

> Der Server muss eingeschaltet sein um zu spielen. Jemand mit Zugriff auf das Dashboard muss den Server starten.

7. **Spielen**  

Server auswählen und `"Join Server"` auswählen.  

> Der Server hat eine Whitelist. Es können nur Spieler mit Erlaubnis spielen. Der Spielername muss von `@llupe18` hinzugefügt werden.

## Custom (ZIP)

In der Zip Datei `Custom Create Mod SMP 1.x.x.zip` sind alle Mods und Shaderpacks als Dateien enthalten.  

Diese können in einem Beliebigen Modloader verwendet werden oder zum Updaten einer vorhandenen Instanz genutzt werden.
